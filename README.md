# JC Data Studio

Plataforma freelance de servicios de datos: dashboards, automatización de Excel, limpieza de datos y SQL.

🔧 **Herramientas gratis incluidas:** dashboard financiero, limpiador de datos (CSV), generador de facturas y constructor de gráficos.
💰 Calculadora de presupuesto instantánea y generador de briefs.

Sitio estático, sin dependencias ni backend. Vanilla JS.

## Correr localmente
```
node serve.js     # http://localhost:8780
```
o simplemente abrir `index.html` en el navegador.

## Personalizar
Editá `config.js`: marca, servicios, precios, links e integraciones. Todo lo demás se adapta solo.
